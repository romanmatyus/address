# RM\Address

## Configuration

### ```config.neon```

```
extensions:
	address: RM\Address\DI\AddressExtension

address:
	google:
		country: sk
		language: sk
```

### ```config.local.neon```

```
address:
	google:
		key: secretKeyFrom300gle
```
